Host it on GitHub Pages (no Python server needed)

  1. Create a new repository named whatever you like and upload these three files.
  2. Keep the included .github/workflows/deploy.yml file so GitHub knows how to publish the site.
  3. Push to main and GitHub Pages will automatically build the static site at https://<username>.github.io/<repo>/ for you to embed anywhere.

Bonus: the four featured games live inside this repo, so GitHub Pages always serves playable content—even if a mirror gets blocked at school.

Tip: On Google Sites, add an "Embed" block and paste the URL of this page or the direct game link. All listed games are HTML5-friendly and hosted on HTTPS mirrors for classroom access.
