Customization tips

Change the accent colors in style.css, update the copy in index.html, or add new entries to the games array inside script.js. The layout automatically updates the card grid, modal, and search.

Need more games?

Use the existing engines as a template. Each game is just a function that receives the canvas context, so cloning a mode is as simple as duplicating one of the engine blocks in script.js.

Lightweight by design

Fonts are served from Google Fonts, but you can swap to a system font stack if you need completely offline deployment. All other assets are plain CSS and JavaScript.
